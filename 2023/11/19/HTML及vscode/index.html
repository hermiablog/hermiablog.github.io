一、基础知识

1、HTML是什么

  • HTML是用来描述网页的语言，又称“超文本标记语言”

    超文本有两层含义

    • 超出了文本的限制：他可以加入图片、视频、音频、链接等
    • 超级链接文本：可以从一个文件跳转到另一个文件，与世界各地主机的文件相连

2、HTML骨架

文档声明DTD：

<!DOCTYPE html>

它并不是HTML标签，文档声明以下的都是HTML标签
作用：告诉浏览器的解析器使用HTML5的标准模式
HTML 4.01有三种声明：严格版、过渡版、框架版

声明语言类型

<html lang="en">

en：英语，zh：中文；网站有多国语言时修改

head标签

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width,initial-scale=1.0">
    <title>Document</title>
</head>

head标签里面是对网页的配置

声明网页字符编码集

<meta charset="UTF-8">

meta是元标签，charset是字符集
在中国字符集有两种类型：”UTF-8”和”gb2312”

字符集 涵盖字符 1个汉字的字节数 适用场景
UTF-8 涵盖全球所有国家、民族的文字和大量图形字符 3 非汉字文字的网页
gb2312(gbk) 收录所有汉字字符、英语，少量韩文、日文、少量图形字符 2 只有汉语和英语的网页
ASCII 只有英文
ISO-8859-1 西欧字符
Unicode 世界通用 它只是符号集，并不是编码方式；具体编码方式包括UTF-8、UTF-16、UTF-32

由上表可以看出，gbk更加节省字节，有利于SEO优化
注意：VScode的设置要与此处一致，否则会乱码。且live serve插件不支持gbk

IE8及以上版本都使用当前版本所对应的最高级标准模式去渲染

<meta http-equiv="X-UA-Compatible" content="IE=edge">

X-UA-Compatible 是IE8的专有meta属性，他告诉IE8采用何种IE版本去渲染网页
最佳兼容模式方案：

<meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1"/>

chrome=1表示将允许站点在使用了谷歌浏览器内嵌框架的客户端渲染

移动端适配

  • 宽度为设备实际宽度，不需要缩放
<meta name="viewport" content="width=device-width,initial-scale=1.0">
  • keywords
<meta name="keywords" content="关键词内容"/>
  • description
<meta name="description" content="描述的内容"/>

网页标题

<title>Document</title>

body标签

里面包括网页的头部、主要内容、页脚各个部分

5、多媒体标签

图片标签img

  • 正确写法：
<img src=" " alt=" " width=" " height=" "/>

pc端以上四要素必须填写
移动端必须填写alt属性，且需要表现出图片的含义

  • alt属性：图片的文本描述
  • width、height属性：图片宽度高度
  • src图片路径：
    • 相对路径：从当前网页出发，找到图片的路径。./是当前目录；../是返回上一级目录。
    • 绝对路径：图片的精确地址

markdown语法：![alt](图片路径)

超级链接a标签

  • 正确写法：
<a herf="链接" title="悬停文本" target="网页跳转方式">网页显示</a>

百度一下

a标签是网页之间跳转的方式
href属性：支持相对属性和绝对属性
title属性：悬停文本
target属性：网页跳转的方式

  • _blank/blank:在新窗口打开网页

  • _self:默认，当前页面跳转

  • _parent:在父窗口中打开

  • _top:在当前窗体打开链接，并替换当前的整个窗体

    • 下面三个标签在网页上看效果一样，但在框架页中效果不同

  • 点击图片跳转网页：

给B处设置id，A处a标签链接为该id，点击A处则会跳转至B处

<a href="#id名">名称</a>

音频标签audio

  • 正确写法1：
<audio controls src="路径" autoplay loop>提示语</audio>

  • controls：显示播放空间

  • src：文件路径

  • 提示语：浏览器不兼容时的文字显示

  • autoplay：自动播放

  • loop：循环播放

  • 正确写法2(加入source标签)：

<audio controls>
    <source src="音频.mp3" type="audio/mpeg" autoplay loop/>
    <source src="音频.ogg" type="audio/ogg" autoplay loop/>
    <source src="音频.wav" type="audio/wav" autoplay loop/>
抱歉，您的浏览器不支持audio标签，请升级您的浏览器哦
</audio>
  • source标签为媒体元素定义媒体资源

它给浏览器提供多种格式的文件，浏览器会选择其支持的格式进行播放；如果都支持则任选一个，默认是第一个。

Format MIME-type 说明
MP3 audio/mpeg 一种音频压缩技术，用来大幅度的降低音频数
Ogg audio/ogg 新的音频压缩技术，完全免费，无专利限制
Wav audio/wav 由微软公司开发，声音质量接近CD
